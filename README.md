# ShoreSquad

## Overview
ShoreSquad is a web application designed to rally your crew for beach cleanups while tracking weather conditions. The platform aims to promote environmental awareness and community involvement in coastal conservation efforts.

## Project Structure
```
ShoreSquad
├── index.html        # Main HTML document
├── css
│   └── styles.css    # Styles for the website
├── js
│   └── app.js        # JavaScript functionality
├── .vscode
│   └── settings.json  # Configuration for Live Server
├── .gitignore        # Files to be ignored by Git
└── README.md         # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd ShoreSquad
   ```
3. Open the project in your preferred code editor.
4. Install any necessary dependencies (if applicable).

## Usage Guidelines
- Open `index.html` in a web browser to view the application.
- Use the interactive features to check weather data and plan beach cleanups.
- Ensure to have a stable internet connection for fetching weather and map data.

## JavaScript Features
- Utilizes the Fetch API to retrieve weather data and map information.
- Implements event listeners for user interactions such as button clicks and form submissions.
- Asynchronous functions are used for efficient data fetching and rendering.
- Local storage is utilized for saving user preferences or recent searches.

## UX Design Principles
- The design is responsive, ensuring usability on both mobile and desktop devices.
- Clear and concise language is used for buttons and labels.
- Keyboard navigation and screen reader support are implemented for accessibility.
- Visual feedback is provided for user actions, including loading indicators and success messages.
- A consistent layout and design are maintained throughout the site for user familiarity.
